## 🎯 Como Usar Esta Documentação

### Para Novos Desenvolvedores:
1. Comece por `01-projeto/` para entender o contexto geral
2. Revise `03-arquitetura/` para compreender a estrutura
3. Explore `04-funcionalidades/` para funcionalidades específicas

### Para Manutenção:
1. Consulte `05-manutencao/` para guias específicos
2. Use `02-migracao/` como referência histórica
3. Verifique `03-arquitetura/` para mudanças estruturais

### Para Implementação de Novas Funcionalidades:
1. Revise `03-arquitetura/` para entender limitações
2. Consulte `04-funcionalidades/` para padrões existentes
3. Siga guias em `05-manutencao/` para implementação
